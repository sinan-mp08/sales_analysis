# 📊 Retail Sales Analytics Dashboard

An interactive Retail Sales Analytics Dashboard built with **Python**, **Streamlit**, and **Plotly** to analyze sales performance, customer behavior, regional trends, and business KPIs.

---

## 🚀 Features

- 📈 Sales Trend Analysis
- 🌍 Regional Sales Performance
- 🛍️ Category-wise Sales Analysis
- 👥 Customer Segmentation
- 🏆 Top Selling Products
- 💰 Revenue & Profit KPIs
- 📊 Interactive Charts & Visualizations
- 🎨 Modern Dark Theme Dashboard

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/sinan-mp08/sales_analysis.git
```

Move into the project

```bash
cd sales_analysis
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
